# 👟 E-commerce de Zapatillas - DSW1

Proyecto de aplicación web de comercio electrónico desarrollado con ASP.NET Core MVC y base de datos SQL Server.

## 📋 Funcionalidades

- 🔐 Sistema de autenticación y registro de usuarios
- 🛒 Catálogo de productos (zapatillas)
- 🛍️ Carrito de compras
- 📦 Gestión de pedidos
- 👤 Panel de usuario
- 📱 Diseño responsive

## 🚀 Ejecutar el proyecto

```bash
# Clonar el repositorio
git clone https://github.com/MonsterCrak/DSW1.Proyecto_Tienda_Zapatillas.Web.git

# Abrir en Visual Studio
# Restore packages
dotnet restore

# Ejecutar
dotnet run
```

## 📁 Estructura del proyecto

```
├── Models/          # Modelos de datos

├── Controllers/     # Controladores MVC
├── Data/            # Contexto de base de datos
```
